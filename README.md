# Заголовок

Сверху вы видите недурный заголовок

\___

## Заголовок поменьше

Этот заголовок конечно на порядок меньше, но тоже неплох

***ВНЕЗАПНЫЙ КОД***

```py
a = 1
print(a)
```

### Заголовок ещё меньше

А этот заголовок самый маленький в этом файле _(после следующего конечно же)_

###### Не забывай

О том, что 6 решёток это максимальное число решёток, которые стоят перед текстом, который является заголовком.  
Ток взгляни какой он мелкий, аж смешно  
Это кстати соответствует уровням заголовков от `<h1>` до `<h6>` в HTML

## Списки

Значится списки у нас до ужаса простые в Markdown (2 основных вида)  
Есть обычный, маркированный спискок:

- Как к примеру
- такой
- список

А есть нумерованый список:

1. Вот
2. такая пошла
3. возня

Также у нас есть вложенные списки:

* Такая
    * штучка
  * дрючка
 
## База символьного форматирования

__Жир__  
_Курсив_  
*Тоже самое*  
**Но звёздочками**  
___И на финалку___  
***Всё вместе***

___ну или если нормально объяснять___

Это *курсив* с помощью звездочек.  
Это _тоже курсив_ с помощью подчеркиваний.

Это **жирный** текст с помощью звездочек.  
Это __тоже жирный__ текст с помощью подчеркиваний.

А это ***жирный курсив*** с помощью звездочек.  
И это ___тоже жирный курсив___ с помощью подчеркиваний.

Формально можно и **_комбинировать_** оба символа (хотя, это ухудшает читабельность текста).

⚠️ Важно: символы выделения должны *примыкать к тексту* без пробелов.

## Описание проекта

Это веб приложение для загадывания желаний интерфейс написан на:

- `HTML`
- `CSS`
- `JavaScript`
- Backend на `Node.js`

## Как запустить

1. Склонируйте репозиторий Git
2. Перейдите в папку проекта
3. Установите зависимости командой `npm install`
4. Запустите сервер командой `npm run start`

## Функционал

- Главная страница
- Страница загадывания желаний
- История желаний
- Настройки
